# React shopping cart

> Create a react shopping cart with redux and redux persist, using fake API for product listing.

## Quick Start

```bash
# Install dependencies
npm install
# Run development server
npm run dev
# Run production build
npm run build
# Run preview mode
npm run preview
# Server runs on http://localhost:8080
```
